# DEVELOPING-A-RECURRENT-NEURAL-NETWORK-MODEL-FOR-STOCK-PREDICTION

## 📈 RNN Stock Price Prediction
### Overview

This project builds a Recurrent Neural Network (RNN/LSTM) model to predict future stock prices using historical time-series data.

### ⚙️ Tech Stack
```
1.Python
2.NumPy, Pandas
3.Matplotlib
4.Scikit-learn
5.TensorFlow / Keras
```

### 🧠 Model
```
1.RNN / LSTM layers for sequence learning
2.Dense layer for final prediction
3.Dropout for overfitting control
```

### 📊 Dataset
```
Historical stock data including:
1.Open
2.Close
3.High
4.Low
5.Volume
```

### 🔄 Workflow
```
1.Load dataset
2.Preprocess & normalize data
3.Create time-series sequences
4.Build and train RNN/LSTM model
5.Predict future stock prices
6.Visualize results
```

### 🚀 Output
```
1.Predicted stock prices
2.Actual vs predicted graph
```
